# Fantasy Mundial 2026

Proyecto base en Next.js 15 + TypeScript + Tailwind CSS.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Qué incluye

- 48 selecciones
- 1.248 jugadores
- Ranking fantasy
- Página de selecciones
- Página de detalle por selección
- Página de jugadores
- Export estático con Next.js
